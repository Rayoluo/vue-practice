# vue_shop前端练手项目

## 安装依赖

```
npm install
```

### 编译和热更新（用于开发环境）

```
npm run serve
```

### 编译并压缩（用于生产环境）

```
npm run build
```

### 检查并修复文件

```
npm run lint
```
